# Welcome to Renku

Renku is a comprehensive toolkit for AI-powered video generation. Create stunning videos with the power of artificial intelligence, from script to screen.

## Getting Started

Documentation is coming soon. In the meantime, you can:

- Check out the [GitHub repository](#github) to explore the source code
- Join our [Discord community](#discord) to ask questions and share your creations

## Features

- **AI-Powered Generation** - Leverage cutting-edge AI models to generate video content
- **Blueprint System** - Define your video structure with flexible blueprints
- **Multiple Providers** - Choose from OpenAI, Replicate, ElevenLabs, and more
- **CLI & SDK** - Integrate video generation into your workflow

Stay tuned for comprehensive documentation covering installation, configuration, and usage guides.
